第一章 引言

1.1 关于 macOS

1.1.1 macOS 的历史与演变

macOS 是苹果公司（Apple Inc.）为其 Mac 系列电脑开发的操作系统。自1984年首次发布以来，macOS 经历了多次重大更新和重命名，反映了技术的进步和用户需求的变化。

  • 初代 Macintosh 操作系统（1984年）：
    • 首次引入图形用户界面（GUI），让用户可以通过图标和窗口进行操作，而非依赖命令行。
  • Mac OS 系列（1984-2001年）：
    • 包括 System 1 到 Mac OS 9，逐步引入多任务处理、虚拟内存、改进的文件管理等功能。
  • Mac OS X / OS X（2001-2016年）：
    • 引入基于 UNIX 的架构，提高系统稳定性和安全性。
    • 采用 Aqua 用户界面，带来更现代化和直观的用户体验。
  • macOS（2016年至今）：
    • 随着 Apple 将操作系统命名统一为 "macOS"（与 iOS、watchOS 等系列保持一致），持续更新和优化。
    • 定期发布新的版本，如 macOS Monterey、macOS Ventura、macOS Sonoma 等，每个版本都带来新的功能和改进。

1.1.2 macOS 的主要特点与优势

macOS 以其独特的设计、强大的功能和生态系统优势，成为许多用户和专业人士的首选操作系统。

  • 用户友好的界面：
    • 简洁、美观的设计，易于导航和使用。
    • 支持触控板手势和多点触控，提高操作效率。
  • 稳定性与安全性：
    • 基于 UNIX 的架构，提供坚实的稳定性和多用户安全模型。
    • 定期的安全更新和内置的防护机制，如 Gatekeeper 和 XProtect，保护系统免受恶意软件和攻击。
  • 深度集成的生态系统：
    • 与其他苹果设备（iPhone、iPad、Apple Watch 等）无缝协作，实现数据同步和跨设备操作。
    • 支持 Continuity、Handoff、AirDrop 等功能，提升整体使用体验。
  • 强大的应用支持：
    • 丰富的应用程序生态系统，包括生产力工具、创意软件、开发环境等。
    • 支持 App Store，方便用户查找、下载和管理应用。
  • 高效的多任务处理：
    • 提供 Mission Control、虚拟桌面（Spaces）等功能，帮助用户高效管理和切换多个任务和窗口。
  • 持续的创新与更新：
    • 定期发布新版本，持续引入新功能和性能优化，保持系统的现代性和竞争力。

1.2 系统需求与兼容性

了解 macOS 的系统需求和兼容性，有助于确保您的设备能够顺利运行最新的操作系统版本，并获得最佳的使用体验。

1.2.1 支持的硬件设备

macOS 的不同版本对硬件有不同的要求。以下是一些通用的系统需求，但具体要求请参考苹果官方网站或每个 macOS 版本的发布说明。

  • 处理器：
    • 64位 Intel 处理器（某些较新的版本可能支持 Apple Silicon，如 M1、M2 芯片）。
  • 内存（RAM）：
    • 至少 4GB 内存，推荐 8GB 或更多，以确保流畅的多任务处理和应用运行。
  • 存储空间：
    • 至少 35.5GB 的可用存储空间用于升级至 macOS Catalina 或更高版本；具体需求因版本而异。
  • 图形：
    • 支持的图形卡，以满足图形界面和图形密集型应用的需求。
  • 显示器：
    • 最低分辨率要求，通常为 1280 x 800 像素或更高。

1.2.2 系统更新与升级

macOS 定期发布系统更新和新版本，以引入新功能、性能优化和安全补丁。了解如何管理系统更新和升级，有助于保持系统的安全性和最新性。

  • 系统更新：

    • 通过“系统偏好设置”中的“软件更新”功能，可以检查和安装最新的系统更新。
    • 系统更新通常包括安全补丁、错误修复和小的功能改进。

  • 系统升级：

    • 每年苹果通常会发布一个新的 macOS 版本（如 macOS Sonoma），提供重大的新功能和改进。
    • 升级前应确保您的设备符合新版本的系统要求，并备份重要数据以防万一。
    • 可以通过“App Store”或“系统偏好设置”中的“软件更新”进行系统升级。

  • 兼容性考虑：

    • 某些旧设备可能无法升级到最新的 macOS 版本。用户需要权衡是否需要最新功能，或继续使用当前版本。
    • 一些第三方应用程序可能需要更新才能与最新的 macOS 版本兼容。确保在升级前检查关键应用的兼容性。

第二章 初始设置

2.1 系统安装与激活

2.1.1 新机首次开机设置

当您首次启动新的 Mac 电脑时，需要完成一系列设置步骤以激活系统并配置基本参数。以下是详细的步骤指南：

  1. 开机步骤

    • 按下电源按钮启动 Mac。
    • 在启动过程中，屏幕会显示苹果标志和加载进度条。

  2. 选择语言和地区

    • 首先，选择您的首选语言。
    • 接着，选择所在的地区或时区，系统会根据选择自动调整时间格式和区域设置。

  3. 连接到网络

    • 选择一个可用的 Wi-Fi 网络并输入密码进行连接。
    • 如果使用有线网络，确保以太网电缆已正确连接。

  4. Apple ID 登录

    • 输入您的 Apple ID 和密码。如果您还没有 Apple ID，可以选择创建一个新的账户。
    • Apple ID 用于访问 App Store、iCloud、iMessage 等苹果服务。

  5. 隐私与安全设置

    • 系统会询问是否启用位置服务、分析数据分享等隐私选项。
    • 根据您的偏好进行选择。

  6. 数据迁移

    • 如果您有旧的 Mac 或 Time Machine 备份，可以选择通过“迁移助理”将数据迁移到新设备。
    • 也可以选择稍后手动迁移数据。

  7. 完成设置

    • 最后，系统会完成剩余的配置并引导您进入桌面环境。

2.1.2 系统恢复与重新安装

在某些情况下，您可能需要恢复系统或重新安装 macOS。以下是常用的方法和步骤：

  1. 使用恢复模式

    • 重新启动 Mac，并在启动时按住 Command (⌘) + R 键，进入恢复模式。
    • 在恢复模式中，您可以选择“重新安装 macOS”、“恢复自 Time Machine 备份”、“使用磁盘工具修复磁盘”等选项。

  2. 通过互联网重新安装

    • 如果本地恢复分区不可用，您可以选择 Option (⌥) + Command (⌘) + R 或 Shift + Option (⌥) + Command (⌘) + R 以通过互联网启动恢复模式。
    • 这将从苹果服务器下载并安装最新兼容的 macOS 版本。

  3. 使用 Time Machine 备份恢复

    • 如果您之前使用 Time Machine 进行了备份，可以在恢复模式中选择“从 Time Machine 备份恢复”。
    • 按照屏幕上的指示选择备份源并开始恢复过程。

  4. 创建可引导的安装器

    • 您可以使用另一个 Mac 创建一个可引导的 macOS 安装器，通常需要一个 USB 闪存驱动器。
    • 下载所需的 macOS 安装程序，通过终端命令创建安装器：
      bash
      sudo /Applications/Install\ macOS\ [版本].app/Contents/Resources/createinstallmedia --volume /Volumes/[驱动器名称]
    • 使用该安装器启动 Mac 并进行安装或恢复。

  5. 常见问题与解决方法

    • 无法进入恢复模式：确保按键组合正确，或尝试通过互联网恢复。
    • 安装失败：检查网络连接，确保有足够的存储空间，或尝试使用可引导安装器。
    • 数据丢失：始终建议在恢复或重新安装前备份重要数据。

2.2 用户账户设置

2.2.1 创建与管理用户账户

macOS 支持多用户环境，允许不同用户拥有各自的设置和文件。以下是创建和管理用户账户的步骤：

  1. 用户账户类型

    • 管理员：具有管理系统设置和其他用户账户的权限。
    • 标准：可以安装软件和更改自己的设置，但无法管理其他用户。
    • 受限（仅适用于儿童账户）：限制访问某些功能和内容，适合家庭共享环境。

  2. 添加新用户

    • 打开“系统偏好设置” > “用户与群组”。
    • 点击左下角的锁图标并输入管理员密码进行解锁。
    • 点击“+”按钮添加新用户。
    • 选择账户类型（管理员、标准、受限），填写用户信息并设置密码。

  3. 修改用户设置

    • 在“用户与群组”中选择要修改的用户。
    • 可以更改用户的全名、密码、头像等信息。
    • 对于管理员账户，可以更改账户类型或赋予管理员权限。

  4. 删除用户账户

    • 在“用户与群组”中选择要删除的用户。
    • 点击“-”按钮删除账户。
    • 系统会询问是否保留用户的主文件夹，选择适当的选项后确认删除。

  5. 更改密码

    • 选择需要更改密码的用户，点击“重置密码”。
    • 输入新密码并确认，必要时设置密码提示。

2.2.2 设置家庭共享

家庭共享允许多个 Apple ID 用户共享购买的内容、订阅和服务，适用于家庭成员之间的使用。以下是设置步骤：

  1. 启用家庭共享

    • 打开“系统偏好设置” > “家庭共享”。
    • 点击“开始使用”，并按照屏幕提示进行设置。

  2. 添加家庭成员

    • 在“家庭共享”设置中，点击“添加家庭成员”。
    • 通过邀请链接或输入家庭成员的 Apple ID 邮箱地址邀请他们加入家庭组。
    • 家庭成员接受邀请后，即可共享内容和服务。

  3. 共享购买的内容

    • App Store：家庭成员可以共享彼此购买的应用、游戏和订阅。
    • iTunes：共享购买的音乐、电影、电视节目和有声读物。
    • Apple Books：共享购买的电子书和有声书。

  4. 共享照片、日历等

    • 共享照片库：启用“iCloud 照片共享”，让家庭成员可以查看和添加照片。
    • 共享日历：创建共享日历，方便家庭成员安排活动和重要事项。
    • 共享提醒事项：共享提醒列表，提高家庭任务管理效率。

  5. 管理家庭共享设置

    • 作为家庭组织者，您可以管理家庭成员的访问权限和共享内容。
    • 可以随时添加或移除家庭成员，调整共享设置以满足家庭需求。

第三章 系统导航

3.1 Finder 使用

3.1.1 Finder 界面介绍

Finder 是 macOS 中用于文件和文件夹管理的核心应用程序，是用户与系统文件进行交互的主要工具。了解 Finder 的界面和功能，有助于您更高效地管理和访问文件。

  • Finder 窗口组成

    • 工具栏：位于窗口顶部，包含常用操作按钮，如返回、前进、新建窗口、显示视图选项等。
    • 侧边栏：显示常用位置和标签，如“收藏夹”、“设备”、“共享”、“标签”等，便于快速访问常用目录。
    • 内容区域：显示当前选定目录中的文件和文件夹，可以根据不同视图模式（图标、列表、列、画廊）进行查看。
    • 状态栏：位于窗口底部，显示当前目录的项数和可用存储空间（可通过“查看”菜单启用或隐藏）。

  • 视图模式

    • 图标视图：以图标形式展示文件和文件夹，适合直观查看内容。
    • 列表视图：以列表形式展示文件，显示更多详细信息，如名称、日期、大小、类型等。
    • 列视图：通过多列方式展示文件层级结构，便于浏览和导航。
    • 画廊视图：主要用于预览多媒体文件，如图片和视频，提供大尺寸预览。

  • Finder 菜单

    • 文件：新建文件夹、打开新窗口、移动到废纸篓等。
    • 编辑：复制、粘贴、剪切、查找等操作。
    • 查看：切换视图模式、显示/隐藏工具栏和侧边栏等。
    • 前往：快速跳转到特定目录，如“应用程序”、“实用工具”等。
    • 窗口：管理多个 Finder 窗口，排列和组织打开的窗口。
    • 帮助：查找 Finder 使用相关的帮助文档和支持资源。

3.1.2 文件与文件夹管理

有效的文件和文件夹管理是提升工作效率的重要环节。以下是一些常用的文件管理操作和技巧：

  1. 创建文件夹

    • 方法一：
      • 在 Finder 中，导航到您希望创建新文件夹的位置。
      • 点击工具栏中的“新建文件夹”按钮，或使用快捷键 Shift + Command + N。
      • 输入文件夹名称后按回车键确认。
    • 方法二：
      • 右键点击空白区域，选择“新建文件夹”。

  2. 重命名文件或文件夹

    • 选中目标文件或文件夹，按 Enter 键，输入新名称后再次按 Enter 键确认。
    • 或者右键点击目标，选择“重命名”。

  3. 复制、剪切与粘贴

    • 复制：选中目标文件或文件夹，按 Command + C。
    • 粘贴：导航到目标位置，按 Command + V 将副本粘贴到新位置。
    • 剪切与粘贴（macOS 默认不支持直接剪切文件，但可以通过复制和删除实现）：
      • 复制文件（Command + C），粘贴到新位置（Command + V），然后删除原文件。

  4. 移动文件或文件夹

    • 直接拖拽：选中目标文件或文件夹，拖动到侧边栏中的目标位置或另一个文件夹中。
    • 使用“移动到”功能：
      • 右键点击目标，选择“移动到”并选择目标位置。

  5. 删除文件或文件夹

    • 选中目标，按 Command + Delete，将其移动到废纸篓。
    • 清空废纸篓：右键点击废纸篓图标，选择“清倒废纸篓”。

  6. 标签与智能文件夹

    • 标签：
      • 右键点击文件或文件夹，选择一种颜色标签进行标记，便于分类和查找。
      • 可以在 Finder 侧边栏中按标签颜色查看所有被标记的项。
    • 智能文件夹：
      • 利用“智能文件夹”功能，创建动态更新的文件集合，基于特定的搜索条件。
      • 在 Finder 菜单中选择“文件” > “新建智能文件夹”，设置搜索条件后保存。

  7. 预览文件

    • 选中目标文件，按 Space 键打开快速预览。
    • 在预览窗口中，可以查看文件内容，支持多种文件格式，如图片、PDF、文档等。

3.2 Dock 和菜单栏

3.2.1 Dock 的自定义与管理

Dock 是 macOS 底部的任务栏，用于快速访问常用应用、文件和文件夹。自定义 Dock 可以提升使用效率和个性化体验。

  • 添加应用到 Dock

    • 打开“应用程序”文件夹，找到目标应用，拖动其图标到 Dock 上合适的位置。
    • 或者在 Launchpad 中找到应用，右键点击选择“选项” > “保留在 Dock 中”。

  • 从 Dock 移除应用

    • 拖动 Dock 上的应用图标离开 Dock，直到出现“移除”提示后松开。
    • 或者右键点击应用图标，选择“选项” > “从 Dock 中移除”。

  • 调整 Dock 的大小和位置

    • 打开“系统偏好设置” > “Dock 与菜单栏”。
    • 在设置窗口中，可以调整 Dock 的大小、放大效果、位置（屏幕底部、左侧或右侧）以及自动隐藏等选项。

  • Dock 分隔线

    • 可以在 Dock 中添加分隔线以组织图标。
    • 打开 Terminal，输入以下命令并按回车：
      bash
      defaults write com.apple.dock persistent-apps -array-add '{"tile-type"="spacer-tile";}'
    • 重新启动 Dock：
      bash
      killall Dock
    • 此时会在 Dock 中添加一个空白分隔线，您可以根据需要多次执行上述命令。

  • 显示最近使用的应用

    • 打开“系统偏好设置” > “Dock 与菜单栏”。
    • 勾选“在 Dock 中显示最近使用的应用”，系统会自动在 Dock 右侧显示最近打开的应用图标。

3.2.2 菜单栏图标与功能

菜单栏 位于屏幕顶部，包含系统和应用的各种状态图标和功能菜单。有效管理菜单栏图标，可以提升系统使用的便利性。

  • 常见的菜单栏图标

    • Apple 菜单：左上角的苹果标志，包含关于本机、系统偏好设置、睡眠、重启、关机等选项。
    • 应用程序菜单：当前活动应用的菜单，包含应用相关的功能选项。
    • 系统图标：如 Wi-Fi、蓝牙、电池、电源、时间显示、通知中心等。
    • 第三方应用图标：安装的第三方应用可能会在菜单栏显示图标，如 Dropbox、Google Drive、音频控制等。

  • 添加和移除菜单栏图标

    • 系统图标：
      • 打开“系统偏好设置” > 对应设置项（如“网络”调整 Wi-Fi 图标显示）。
      • 勾选“在菜单栏中显示”选项。
    • 第三方应用：
      • 某些应用在安装后会自动添加菜单栏图标，可以在应用的设置中进行管理和调整。
      • 若要移除，通常需要在应用内设置或通过拖拽图标到废纸篓。

  • 使用第三方工具管理菜单栏

    • Bartender、iStat Menus 等第三方工具，可以帮助更好地管理和组织菜单栏图标，隐藏不常用的图标，保持菜单栏整洁。

  • 调整菜单栏显示内容

    • 打开“系统偏好设置” > “Dock 与菜单栏”。
    • 在左侧列表中选择需要调整的项目，设置其显示方式和内容。

3.3 Spotlight 搜索

3.3.1 使用 Spotlight 快速查找

Spotlight 是 macOS 内置的强大搜索工具，可以快速查找文件、应用、邮件、联系人、网页信息等。

  • 启动 Spotlight

    • 点击屏幕右上角的放大镜图标，或按下快捷键 Command + Space。

  • 基本搜索

    • 输入关键词，Spotlight 会实时显示匹配的结果，包括应用、文件、文件夹、邮件、联系人等。
    • 使用键盘方向键浏览结果，按下回车键打开选定的项目。

  • 计算器与转换

    • 在搜索栏输入数学表达式，如 50 * 12，Spotlight 会即时显示计算结果。
    • 支持货币、度量单位、时间等多种转换，如 100 USD to EUR、100 km to miles。

  • 快速打开应用

    • 输入应用名称，如“Safari”，按回车键即可快速启动应用。

  • 查找系统设置

    • 输入设置项名称，如“蓝牙”，Spotlight 会显示“系统偏好设置”中的相关选项，点击即可直接跳转。

  • 网络搜索

    • 输入问题或关键词，Spotlight 会显示相关的网页搜索结果，点击即可在默认浏览器中打开。

3.3.2 Spotlight 设置与优化

优化 Spotlight 的设置，可以提升搜索的效率和相关性，确保您获得最准确的搜索结果。

  • 访问 Spotlight 设置

    • 打开“系统偏好设置” > “聚焦”（Spotlight）。

  • 调整搜索类别

    • 在“搜索结果”选项卡中，勾选或取消勾选不同的搜索类别，如“应用程序”、“文档”、“邮件”、“消息”等，根据需要定制搜索范围。

  • 调整搜索快捷键

    • 在“键盘”设置中，可以自定义 Spotlight 的快捷键，适应个人使用习惯。

  • 排除特定文件夹

    • 在“系统偏好设置” > “聚焦” > “隐私”选项卡中，添加不希望被 Spotlight 索引的文件夹或磁盘，确保搜索结果中不包含这些位置的内容。

  • 重建 Spotlight 索引

    • 如果发现 Spotlight 搜索结果不准确或缺失，可以通过“系统偏好设置” > “聚焦” > “隐私”添加并移除整个磁盘来强制重建索引。
    • 具体步骤：
      1. 打开“系统偏好设置” > “聚焦” > “隐私”。
      2. 点击“+”按钮，添加您的启动磁盘（通常是“Macintosh HD”）。
      3. 选中刚添加的磁盘，点击“-”按钮将其移除，系统将开始重新索引。

  • 使用关键词优化搜索

    • 利用特定关键词和操作符，可以更精确地进行搜索。
      • 文件类型搜索：如 kind:pdf 查找所有 PDF 文件。
      • 日期搜索：如 date:today 查找今天修改的文件。
      • 所有人搜索：如 author:张三 查找特定作者的文件。

第四章 应用程序管理

4.1 安装与卸载应用

4.1.1 从 App Store 安装应用

App Store 是 macOS 官方提供的应用分发平台，提供了丰富的免费和付费应用，确保应用的安全性和兼容性。

  1. 打开 App Store

    • 点击 Dock 上的 App Store 图标，或通过 Launchpad 搜索并打开 App Store。

  2. 浏览和搜索应用

    • 首页推荐：展示精选应用、畅销榜和编辑推荐。
    • 分类浏览：在侧边栏选择不同类别，如“生产力”、“教育”、“游戏”等。
    • 搜索功能：使用右上角的搜索栏输入应用名称、关键词或开发者名称进行搜索。

  3. 查看应用详情

    • 点击应用图标，进入应用详情页面。
    • 查看应用描述、功能介绍、用户评价、截图和视频预览等信息。

  4. 下载与安装

    • 点击 获取 按钮（对于免费应用）或显示价格的按钮（对于付费应用）。
    • 如果需要购买，系统会提示输入 Apple ID 密码或使用 Touch ID 进行验证。
    • 下载完成后，应用会自动安装，并在 Launchpad 中显示。

  5. 更新应用

    • 打开 App Store，点击左侧的 更新 标签。
    • 查看可用更新，点击 更新 按钮进行更新，或点击 全部更新 一次性更新所有应用。

4.1.2 手动安装第三方应用

除了 App Store，您还可以从第三方网站下载并安装应用。以下是手动安装应用的步骤：

  1. 下载应用

    • 访问应用开发者的官方网站或其他可信的软件下载平台。
    • 下载适用于 macOS 的应用安装包，通常为 .dmg（磁盘映像文件）、.pkg（安装包）或压缩文件（如 .zip）。

  2. 安装应用

    • .dmg 文件：
      1. 双击 .dmg 文件，挂载磁盘映像。
      2. 在打开的窗口中，通常会显示应用图标和 应用程序 文件夹的快捷方式。
      3. 将应用图标拖动到 应用程序 文件夹中完成安装。
      4. 弹出磁盘映像，卸载挂载的驱动器。
    • .pkg 文件：
      1. 双击 .pkg 文件，启动安装向导。
      2. 按照屏幕提示完成安装过程。
    • 压缩文件：
      1. 双击压缩文件（如 .zip），解压缩内容。
      2. 通常会得到一个应用程序图标，拖动到 应用程序 文件夹中。

  3. 首次打开应用

    • 第一次打开非 App Store 安装的应用时，系统可能会提示“无法打开，因为它来自不明开发者”。
    • 解决方法：
      1. 打开 系统偏好设置 > 安全性与隐私 > 通用。
      2. 在下方看到相关提示，点击 仍然打开。
      3. 或者，右键点击应用图标，选择 打开，然后确认打开。

4.1.3 卸载应用的方法

卸载应用可以帮助您释放存储空间并保持系统整洁。以下是几种常用的卸载方法：

  1. 通过 Launchpad 卸载应用

    • 打开 Launchpad。

    • 找到要卸载的应用，点击并按住应用图标，直到图标开始晃动。

    • 点击应用图标左上角出现的 × 按钮，确认删除。

    注意：此方法仅适用于通过 App Store 安装的应用。

  2. 通过 Finder 删除应用

    • 打开 Finder，导航到 应用程序 文件夹。

    • 找到要删除的应用，右键点击选择 移到废纸篓，或拖动应用图标到 Dock 上的废纸篓图标。

    • 清空废纸篓以永久删除应用。

    提示：有些应用可能在其他位置存储配置文件或缓存，您可以使用第三方工具（如 AppCleaner）进行彻底卸载。

  3. 使用第三方卸载工具

    • AppCleaner、CleanMyMac 等第三方工具可以帮助您全面删除应用及其相关文件。
    • 下载并安装所选工具，按照指示选择要卸载的应用，进行彻底删除。

macOS 提供了应用权限管理功能，确保应用只能访问授权的系统资源和用户数据。

  4. 监控和管理应用权限

    • 定期检查和更新应用权限，确保只有可信的应用获得必要的访问权限。

6.1 文件和文件夹操作

有效的文件和文件夹管理是提升工作效率和组织能力的关键。macOS 提供了丰富的工具和功能，帮助用户轻松创建、复制、移动、删除和组织文件与文件夹。

6.1.1 创建、复制、移动与删除

创建文件夹

  1. 通过 Finder 创建

    • 打开 Finder。
    • 导航到您希望创建新文件夹的位置。
    • 在菜单栏中选择 文件 > 新建文件夹，或使用快捷键 Shift + Command + N。
    • 输入文件夹名称后按 Enter 键确认。

  2. 通过右键菜单创建

    • 在目标位置的空白区域右键点击。
    • 选择 新建文件夹。
    • 输入文件夹名称并确认。
复制文件或文件夹

  1. 使用快捷键

    • 选中要复制的文件或文件夹。
    • 按 Command + C 复制。
    • 导航到目标位置，按 Command + V 粘贴。

  2. 拖拽复制

    • 选中要复制的文件或文件夹。
    • 按住 Option 键，拖动文件到目标位置。
    • 松开鼠标按钮，文件将被复制到新位置。
移动文件或文件夹

  1. 使用拖拽移动

    • 选中要移动的文件或文件夹。
    • 拖动到目标位置，松开鼠标按钮，文件将被移动。

  2. 使用剪切和粘贴

    • macOS 默认不支持直接剪切文件，但可以通过复制和删除实现：
      • 复制文件（Command + C）。
      • 粘贴到目标位置（Command + V）。
      • 删除原文件（Command + Delete）。
删除文件或文件夹

  1. 移动到废纸篓

    • 选中要删除的文件或文件夹。
    • 按 Command + Delete，或右键点击选择 移到废纸篓。

  2. 永久删除

    • 清空废纸篓：右键点击 废纸篓 图标，选择 清倒废纸篓。
    • 或者，按 Option + Command + Delete 直接清空废纸篓。
恢复删除的文件
  • 打开 废纸篓，找到要恢复的文件或文件夹。
  • 右键点击选择 放回原处，文件将恢复到删除前的位置。

6.1.2 标签与智能文件夹

使用标签

  1. 为文件或文件夹添加标签

    • 右键点击目标文件或文件夹。
    • 在弹出的菜单中选择一种颜色标签，或点击 标签 选项并选择颜色。

  2. 通过 Finder 侧边栏查看标签

    • 在 Finder 的侧边栏中，可以看到不同颜色的标签。
    • 点击标签颜色，显示所有被标记的文件和文件夹。

  3. 自定义标签名称

    • 打开 Finder，在菜单栏选择 Finder > 偏好设置。
    • 点击 标签 标签页，可以重命名、添加或删除标签。
创建智能文件夹

智能文件夹是基于特定搜索条件自动汇总文件和文件夹的动态文件集合。

  1. 创建智能文件夹

第七章 多任务与窗口管理

7.1 Mission Control

Mission Control 是 macOS 提供的强大工具，允许用户快速查看和管理所有打开的窗口、桌面和全屏应用。通过 Mission Control，您可以更高效地切换任务、组织工作空间，并保持桌面的整洁。

7.1.1 使用 Mission Control 查看所有窗口

  1. 启动 Mission Control

    • 快捷键：按下 Control (⌃) + 上箭头 (↑) 键。
    • 触控板手势：使用四指向上滑动。
    • Dock 图标：点击 Dock 上的 Mission Control 图标（通常是一个带有多个窗口的图标）。

  2. 界面介绍

    • 顶部区域：显示所有虚拟桌面（Spaces）和全屏应用。
    • 中央区域：显示当前所有打开的窗口，按照应用分类排列。
    • 底部区域：显示应用程序的图标，允许快速访问和管理。

  3. 浏览和选择窗口

    • 在 Mission Control 界面中，使用鼠标或触控板浏览所有打开的窗口。
    • 点击任意窗口即可将其切换为当前活动窗口。
    • 使用键盘方向键和 Tab 键在窗口之间导航，按 Enter 键选择。

  4. 管理虚拟桌面

    • 在顶部区域，您可以看到当前存在的所有虚拟桌面。
    • 点击 + 按钮可以添加新的虚拟桌面。
    • 拖动窗口到不同的虚拟桌面，或将其移动到新的桌面中。

  5. 关闭窗口

    • 在 Mission Control 中，悬停在某个窗口上方，会显示窗口的关闭按钮（红色的“X”），点击即可关闭该窗口。

7.1.2 创建与管理虚拟桌面

虚拟桌面（Spaces） 允许您在多个工作空间之间切换，每个桌面可以有不同的应用和窗口，帮助您更好地组织和管理工作任务。

  1. 创建新的虚拟桌面

    • 启动 Mission Control（Control + 上箭头）。
    • 在顶部区域，点击右上角的 + 按钮，创建一个新的虚拟桌面。
    • 新创建的桌面会出现在顶部区域，您可以为其命名以便区分（需要第三方工具，如 TotalSpaces）。

  2. 切换虚拟桌面

    • 快捷键：按下 Control (⌃) + 左箭头 (←) 或 Control (⌃) + 右箭头 (→) 键，在虚拟桌面之间切换。
    • 触控板手势：使用四指向左或向右滑动。

  3. 移动窗口到不同的虚拟桌面

    • 启动 Mission Control，拖动窗口到目标虚拟桌面。
    • 或者，右键点击应用图标，在弹出的菜单中选择 选项 > 分配到，然后选择目标桌面。

  4. 删除虚拟桌面

    • 启动 Mission Control，悬停在要删除的虚拟桌面上，点击左上角的 X 按钮。
    • 关闭虚拟桌面后，桌面上的窗口会自动移动到剩余的桌面中。

  5. 组织虚拟桌面

    • 将相关的应用和窗口分配到特定的虚拟桌面，例如，一个桌面用于工作，另一个用于娱乐，提升工作效率和使用体验。

7.2 分屏与全屏应用

分屏模式 和 全屏应用 是 macOS 提供的窗口管理功能，允许用户在屏幕上同时显示多个应用，或将单个应用全屏显示，提升多任务处理能力和专注度。

7.2.1 分屏模式的使用

分屏模式（Split View） 允许您将两个应用并排显示在全屏模式下，方便同时进行多任务处理。

  1. 启用分屏模式

    • 打开您希望使用分屏模式的第一个应用窗口。
    • 点击窗口左上角的 绿色全屏按钮 并按住，直到窗口缩小并显示屏幕的左右两个区域。
    • 将窗口拖动到屏幕的左侧或右侧，释放鼠标，窗口将占据屏幕的一半。

  2. 选择第二个应用

    • 在剩余的半屏区域中，选择另一个应用窗口进行分屏显示。
    • 点击所选应用窗口，它将自动占据屏幕的另一半。

  3. 调整窗口大小

    • 分屏模式下，屏幕中间会出现一个可拖动的分隔条。
    • 拖动分隔条可以调整两个应用窗口的大小比例，满足不同的使用需求。

  4. 退出分屏模式

    • 将鼠标悬停在分隔条上，点击出现的 全屏按钮，退出分屏模式。
    • 或者，按下 Esc 键退出全屏应用。

  5. 使用快捷键管理分屏

    • 使用 Control + 上箭头 启动 Mission Control，然后将应用拖动到分屏模式中。

7.2.2 全屏应用的管理

将应用置于全屏模式可以帮助您专注于当前任务，减少干扰，提升工作效率。

  1. 启用全屏模式

    • 打开目标应用窗口。
    • 点击窗口左上角的 绿色全屏按钮，应用将切换到全屏模式，占据整个屏幕。

  2. 切换全屏应用

    • 使用 Control (⌃) + 左箭头 (←) 或 Control (⌃) + 右箭头 (→) 键在多个全屏应用和虚拟桌面之间切换。
    • 或者，通过触控板的四指向左或向右滑动手势进行切换。

  3. 退出全屏模式

    • 将鼠标移至屏幕顶部，出现菜单栏后，点击窗口左上角的 绿色全屏按钮。
    • 或者，按下 Esc 键退出全屏模式。

  4. 管理全屏应用

    • 在 Mission Control 中，可以看到所有全屏应用，点击任意一个即可切换到对应的全屏应用。

  5. 多显示器支持

    • 在多显示器环境下，每个显示器都可以独立管理全屏应用。
    • 将应用置于全屏模式时，它仅占据当前显示器，其他显示器上的应用和桌面不受影响。

7.3 快速切换与任务管理

macOS 提供了多种快捷方式和工具，帮助用户在多个任务和应用之间快速切换，提升工作效率。

  1. 应用切换

    • 快捷键：按下 Command (⌘) + Tab 键，打开应用切换器。

  2. 窗口切换

    • 在 Dock 上点击应用图标，即可快速切换到该应用。
    • 右键点击 Dock 上的应用图标，选择 切换到 或 全部窗口，查看所有打开的窗口。

  5. Mission Control 快速切换

    • 启动 Mission Control，通过拖动和选择窗口、虚拟桌面进行快速切换。
    • 使用 Mission Control 中的虚拟桌面和全屏应用进行任务组织。

  6. Spotlight 快速启动

    • 按下 Command + Space 打开 Spotlight，输入应用名称，按 Enter 键快速启动应用。
    • 例如，输入 "Safari"，然后按 Enter 启动 Safari 浏览器。

8.1 系统安全设置

macOS 提供了一系列内置的安全功能，旨在保护您的系统免受恶意软件、未经授权的访问和数据泄露的威胁。以下是关键的系统安全设置及其配置方法。

8.1.1 防火墙与文件保险箱

启用防火墙

防火墙帮助您监控和控制进出 Mac 的网络流量，阻止潜在的恶意连接。

  1. 打开防火墙设置

    • 打开 系统偏好设置 > 安全性与隐私 > 防火墙。
    • 如果防火墙未开启，点击左下角的锁图标并输入管理员密码进行解锁。
    • 点击 开启防火墙 按钮。

  2. 配置防火墙选项

    • 点击 防火墙选项… 按钮，进行更详细的设置。
    • 启用隐身模式：勾选此选项可使您的 Mac 在未连接到任何网络时隐藏其 IP 地址，增强隐私保护。
    • 允许特定应用通过防火墙：
      • 点击 + 按钮，添加您信任的应用，允许它们通过防火墙进行通信。
      • 选中不需要通过防火墙的应用，点击 - 按钮将其移除。
启用文件保险箱（FileVault）

文件保险箱使用全盘加密技术，保护您的数据在设备丢失或被盗时不被未授权访问。

  1. 开启文件保险箱

    • 打开 系统偏好设置 > 安全性与隐私 > 文件保险箱。
    • 点击 开启文件保险箱 按钮。
    • 选择一个解锁文件保险箱的方式：
      • 使用密码：确保选择一个强密码，并牢记该密码。
      • 恢复密钥：备份恢复密钥，以便在忘记密码时恢复数据。

  2. 开始加密

    • 确认设置后，文件保险箱将开始加密您的磁盘。加密过程可能需要一些时间，取决于磁盘的大小和数据量。
    • 在加密过程中，您可以继续正常使用 Mac。

  3. 管理文件保险箱

    • 返回 文件保险箱 设置，可以随时查看加密状态。
    • 点击 显示恢复密钥 或 更改密码 按钮，进行相关管理操作。

8.1.2 Gatekeeper 与应用安全

Gatekeeper 是 macOS 的一项安全功能，旨在确保您只运行来自可靠来源的应用，防止恶意软件的入侵。

  1. 配置 Gatekeeper 设置

    • 打开 系统偏好设置 > 安全性与隐私 > 通用。
    • 在 允许从以下位置下载的应用程序 部分，您可以选择以下选项：
      • App Store：仅允许从 Mac App Store 下载的应用程序运行。
      • App Store 和被认可的开发者：允许从 Mac App Store 和具有苹果认证的开发者下载的应用程序运行。
    • 高级设置：
      • 点击 仍然允许 按钮，以临时允许某个特定的应用程序运行。

  2. 验证应用来源

    • 下载应用时，确保应用来自可信的开发者或官方网站。
    • 通过 应用程序 文件夹中的 显示简介 功能，查看应用的开发者信息。

  3. 处理无法打开的应用

    • 如果尝试打开一个未被 Gatekeeper 认可的应用，系统会提示“无法打开，因为它来自不明开发者”。
    • 解决方法：
      1. 打开 系统偏好设置 > 安全性与隐私 > 通用。
      2. 在下方看到相关提示，点击 仍然打开。
      3. 或者，右键点击应用图标，选择 打开，然后在弹出的对话框中确认打开。

8.2 数据备份

定期备份您的数据是确保信息安全的重要措施。macOS 提供了多种备份解决方案，帮助您在数据丢失或系统故障时恢复重要文件。

8.2.1 Time Machine 备份设置

Time Machine 是 macOS 内置的备份工具，能够自动备份您的系统和数据，便于快速恢复。

  1. 配置 Time Machine

    • 连接一个外部存储设备（如 USB 硬盘、Time Capsule）。
    • 打开 系统偏好设置 > Time Machine。
    • 点击 选择备份磁盘…，选择已连接的外部存储设备。
    • 勾选 自动备份，Time Machine 将定期自动进行备份。

  2. 管理备份

    • 在 Time Machine 界面，可以查看备份的状态和历史记录。
    • 点击 选项… 按钮，配置不需要备份的文件或文件夹。
    • 通过 排除项目 添加不需要备份的内容，节省存储空间。

  3. 恢复数据

    • 打开 Time Machine，进入备份浏览界面。
    • 使用时间轴选择需要恢复的文件或文件夹。
    • 选择目标文件，点击 恢复 按钮，将其恢复到原位置或指定的新位置。

8.2.2 iCloud 备份与同步

iCloud 提供了云端存储和同步服务，允许您在多个苹果设备之间无缝访问和备份数据。

  1. 配置 iCloud 备份

    • 打开 系统偏好设置 > Apple ID > iCloud。
    • 登录您的 Apple ID，如果尚未登录，输入 Apple ID 和密码。
    • 勾选需要同步和备份到 iCloud 的项目，如 iCloud Drive、照片、邮件、联系人、日历、备忘录 等。

  2. 使用 iCloud Drive

    • 启用 iCloud Drive，将文件存储在云端，支持跨设备访问。
    • 在 Finder 侧边栏中点击 iCloud Drive，查看和管理存储的文件。

  3. 管理 iCloud 存储

    • 在 iCloud 设置中，点击 管理 按钮，查看存储使用情况。
    • 删除不需要的备份、文件或邮件，释放存储空间。
    • 如需更多存储空间，可以选择升级到更高的存储计划。

8.2.3 第三方备份解决方案

除了 Time Machine 和 iCloud，您还可以使用第三方备份工具，提供更多定制化的备份选项和功能。

  1. Carbon Copy Cloner

    • 简介：Carbon Copy Cloner 是一款功能强大的备份工具，支持创建可引导的克隆备份，快速恢复系统。

      • 定期自动备份。
      • 支持增量备份，节省存储空间。
      • 创建可引导的备份，快速恢复系统故障。

  2. SuperDuper!

    • 简介：SuperDuper! 提供简单易用的备份和克隆功能，适合个人用户。
    • 功能：
      • 一键备份和恢复。
      • 支持定时备份任务。
      • 创建可引导的克隆备份。

  3. Backblaze

    • 简介：Backblaze 是一款在线备份服务，提供无限的云存储空间，自动备份您的数据到云端。
    • 功能：
      • 自动备份，无需手动操作。
      • 数据加密，确保隐私安全。
      • 随时随地访问和恢复备份数据。

8.3 隐私设置

macOS 提供了丰富的隐私设置，允许您控制应用程序对个人数据和系统资源的访问权限，确保您的隐私不被侵犯。

8.3.1 管理应用权限

  1. 访问隐私设置

    • 打开 系统偏好设置 > 安全性与隐私 > 隐私 标签页。

  2. 调整应用权限

    • 在左侧列表中选择不同的权限类别，如 位置服务、联系人、日历、照片、麦克风、相机 等。
    • 在右侧列表中，勾选或取消勾选允许访问该类别的应用。
    • 提示：更改某些权限可能需要点击左下角的锁图标并输入管理员密码进行解锁。

  3. 审查和更新权限

    • 定期检查哪些应用拥有哪些权限，撤销不需要或不信任的应用权限，增强系统安全性。

8.3.2 位置服务

位置服务允许应用程序访问您的地理位置信息，提升应用功能的同时也可能带来隐私风险。

  1. 配置位置服务

    • 打开 系统偏好设置 > 安全性与隐私 > 隐私 > 位置服务。
    • 勾选 启用位置服务，并选择允许访问位置服务的应用。

  2. 管理位置权限

    • 在列表中，针对每个应用，选择 始终、使用应用期间 或 不允许 访问位置服务。
    • 提示：仅允许信任的应用访问位置服务，避免不必要的隐私泄露。

  3. 系统服务的定位

    • 滚动到底部，可以管理系统服务的定位权限，如 查找我的 Mac、时间和日期设置 等。
    • 根据需要启用或禁用特定的系统服务定位权限。

8.4 用户账户安全

确保用户账户的安全性，有助于防止未授权的访问和保护个人数据。

8.4.1 设置强密码

  1. 更改密码

    • 打开 系统偏好设置 > 用户与群组。
    • 选择当前用户，点击 更改密码… 按钮。
    • 输入旧密码、新密码及确认新密码，确保密码复杂且难以猜测。

  2. 使用密码提示

    • 在设置新密码时，添加一个有助于记忆的密码提示，但避免使用过于明显的信息。

  3. 启用密码过期

    • 虽然 macOS 默认不强制密码定期更换，但建议定期手动更改密码，提升账户安全性。

8.4.2 双因素认证

双因素认证（2FA）为您的 Apple ID 提供额外的安全层，防止未经授权的访问。

  1. 启用双因素认证

    • 打开 系统偏好设置 > Apple ID > 密码与安全性。
    • 点击 启用双因素认证，并按照屏幕上的指示完成设置。
    • 需要一个受信任的设备或电话号码来接收验证码。

  2. 使用双因素认证

    • 每次登录 Apple ID 时，除了输入密码，还需输入发送到受信任设备的验证码。
    • 这有效防止他人通过窃取密码访问您的账户。

  3. 管理受信任的设备和电话号码

    • 在 密码与安全性 设置中，查看和管理受信任的设备和电话号码。
    • 添加或移除受信任的电话号码，确保仅使用您信任的设备接收验证码。

8.5 防范恶意软件

虽然 macOS 系统相对安全，但仍需采取措施防范恶意软件和网络攻击，确保系统和数据的完整性。

8.5.1 使用防病毒软件

  1. 选择可靠的防病毒软件

    • 推荐使用市场上知名的防病毒软件，如 Malwarebytes for Mac、Avast Security、Norton 360 等。
    • 这些软件提供实时扫描、定期更新病毒库和恶意软件清除功能。

  2. 安装和配置防病毒软件

    • 下载防病毒软件的安装包，按照安装向导完成安装。
    • 启用实时保护和自动扫描功能，定期进行系统全盘扫描，检测和清除潜在的威胁。

  3. 保持防病毒软件更新

    • 定期检查和安装防病毒软件的更新，确保病毒库和防护机制保持最新。

8.5.2 安全的浏览习惯

  1. 使用安全的浏览器

    • 使用 macOS 内置的 Safari 浏览器，或其他安全性较高的浏览器，如 Mozilla Firefox、Google Chrome。
    • 确保浏览器保持最新版本，获得最新的安全补丁和功能。

  2. 启用浏览器安全功能

    • 在浏览器设置中，启用 反钓鱼保护、阻止弹出窗口、限制第三方 Cookie 等安全功能。
    • 使用 浏览器扩展 增强安全性，如广告拦截器、防追踪插件等。

  3. 避免访问可疑网站

    • 避免点击不明来源的链接和附件，尤其是来自未知或不信任的电子邮件和消息。
    • 使用 HTTPS 加密连接，确保数据传输的安全性。

  4. 定期清理浏览数据

    • 定期清理浏览器的缓存、Cookie 和浏览历史，减少隐私泄露的风险。

8.6 恢复安全事件

在遭遇安全事件（如数据泄露、系统感染）时，及时采取措施恢复系统和保护数据至关重要。

8.6.1 重置密码

  1. 重置 Apple ID 密码

    • 打开 系统偏好设置 > Apple ID > 密码与安全性。
    • 点击 更改密码，按照提示使用双因素认证重置密码。
    • 如果无法访问受信任的设备或电话号码，可以通过 Apple 支持 获取帮助。

  2. 重置用户账户密码

    • 如果忘记了 Mac 用户账户密码，可以通过 恢复模式 进行重置：
      1. 重新启动 Mac，并在启动时按住 Command (⌘) + R 键，进入恢复模式。
      2. 在 实用工具 菜单中选择 终端。
      3. 输入 resetpassword 并按回车键，启动密码重置工具。
      4. 按照屏幕提示选择用户账户，输入新密码并确认。

8.6.2 恢复备份

  1. 使用 Time Machine 恢复

    • 连接 Time Machine 备份磁盘。
    • 打开 系统偏好设置 > Time Machine，点击 进入 Time Machine。
    • 浏览备份历史，选择需要恢复的文件或系统状态，点击 恢复。

  2. 使用 iCloud 恢复

    • 登录您的 Apple ID，访问 iCloud.com。
    • 下载和恢复同步到 iCloud 的文件和数据。

  3. 使用第三方备份工具恢复

    • 根据您使用的第三方备份工具（如 Carbon Copy Cloner、SuperDuper!），按照软件提供的恢复流程进行数据恢复。

9.1 Terminal 使用

Terminal 是 macOS 内置的命令行界面，基于 UNIX，可以让用户通过输入命令来执行各种系统操作和任务。掌握 Terminal 的使用，可以大幅提升操作效率，进行高级系统配置和自动化任务。

9.1.1 基础命令介绍

以下是一些常用的 Terminal 基础命令，帮助您开始使用命令行：

  1. 导航文件系统

    • pwd（Print Working Directory）：显示当前所在的目录路径。
      bash
      pwd
    • ls（List）：列出当前目录中的文件和文件夹。
      bash
      ls
    • cd（Change Directory）：切换到指定的目录。
      bash
      cd /路径/到/目录
      例如，切换到桌面：
      bash
      cd ~/Desktop

  2. 文件和文件夹操作

    • mkdir（Make Directory）：创建新的文件夹。
      bash
      mkdir 新文件夹名称
    • touch：创建新的空文件。
      bash
      touch 文件名.txt
    • cp（Copy）：复制文件或文件夹。
      bash
      cp 源文件 目标路径
      例如，复制文件到桌面：
      bash
      cp 文件.txt ~/Desktop/
    • mv（Move）：移动或重命名文件或文件夹。
      bash
      mv 源文件 目标路径
      例如，重命名文件：
      bash
      mv 旧文件名.txt 新文件名.txt
    • rm（Remove）：删除文件或文件夹。
      • 删除文件：
        bash
        rm 文件名.txt
      • 删除文件夹及其内容：
        bash
        rm -r 文件夹名称

  3. 查看文件内容

    • cat：显示文件内容。
      bash
      cat 文件名.txt
    • less：分页查看文件内容，适用于较大的文件。
      bash
      less 文件名.txt
    • nano 或 vim：使用文本编辑器编辑文件。
      bash
      nano 文件名.txt
      或
      bash
      vim 文件名.txt

  4. 系统信息

    • top：实时显示系统资源使用情况。
      bash
      top
    • df：显示磁盘空间使用情况。
      bash
      df -h
    • du：显示文件或文件夹的磁盘使用量。
      bash
      du -sh 文件或文件夹名称

9.1.2 常用 Terminal 操作

  1. 使用 Homebrew 安装软件

    Homebrew 是 macOS 上流行的包管理器，方便安装和管理各种开源软件和命令行工具。

    • 安装 Homebrew
      bash
      /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    • 安装软件 例如，安装 wget：
      bash
      brew install wget
    • 更新 Homebrew 和已安装的软件
      bash
      brew update
brew upgrade
    • 卸载软件
      bash
      brew uninstall wget

  2. 文件权限管理

    • 查看文件权限
      bash
      ls -l 文件名
      输出示例：
      css
      -rw-r--r--  1 用户  组  0 1 1 12:00 文件名.txt
    • 更改文件权限
      bash
      chmod 权限 文件名
      例如，赋予所有人执行权限：
      bash
      chmod +x 脚本.sh
    • 更改文件所有者
      bash
      sudo chown 新所有者:新组 文件名
      例如，将文件所有者更改为 john：
      bash
      sudo chown john:staff 文件名.txt

  3. 网络操作

    • Ping 测试
      bash
      ping example.com
      按 Ctrl + C 停止测试。
    • 查看本机 IP 地址
      bash
      ifconfig
    • 下载文件 使用 wget 或 curl 下载文件：
      bash
      wget http://example.com/file.zip
      或

9.2 Automator 和快捷指令

Automator 和 快捷指令（Shortcuts） 是 macOS 提供的自动化工具，允许用户创建自定义的工作流程和自动化任务，简化日常操作，提升工作效率。

9.2.1 Automator 使用

Automator 是 macOS 内置的自动化工具，提供了图形化的界面，用户无需编写代码即可创建复杂的自动化任务。

9.3 高级自动化与脚本

除了 Automator 和快捷指令，macOS 还支持使用脚本语言（如 AppleScript 和 Python）进行更复杂和定制化的自动化任务。

9.3.1 使用 AppleScript

AppleScript 是 macOS 的一种脚本语言，专为自动化任务和控制应用程序设计。

  1. 启动 Script Editor

    • 打开 Launchpad，搜索 Script Editor 并启动。
    • 或者，通过 应用程序 > 实用工具 文件夹找到并打开 Script Editor。

  2. 编写 AppleScript 脚本

    • 在 Script Editor 中，新建一个脚本文件。
    • 编写脚本代码，例如，自动打开 Safari 并访问特定网站：
      applescript
      tell application "Safari"
    activate
    open location "https://www.apple.com"
end tell

  3. 运行脚本

    • 点击 Script Editor 窗口顶部的 运行 按钮，执行脚本。
    • 确认脚本按预期运行，Safari 应自动打开并访问 Apple 官网。

  4. 保存和使用脚本

    • 选择 文件 > 保存，将脚本保存为 .scpt 文件。
    • 您可以将脚本保存为应用程序，双击运行，或将其集成到 Automator 工作流程和快捷指令中。

  5. 示例 AppleScript 脚本

    自动整理下载文件

    applescript
    tell application "Finder"
    set downloadFolder to folder "Downloads" of home
    set documentsFolder to folder "Documents" of home
    set pdfFiles to every file of downloadFolder whose name extension is "pdf"
    repeat with aFile in pdfFiles
        move aFile to folder "PDFs" of documentsFolder
    end repeat
end tell

9.3.2 使用 Python 进行自动化

macOS 预装了 Python 解释器（根据版本不同，可能是 Python 2 或 Python 3）。通过 Python，您可以编写脚本执行复杂的自动化任务。

  1. 检查 Python 版本

    bash
    python --version

    或

    bash
    python3 --version

  2. 编写 Python 脚本

    • 使用文本编辑器（如 TextEdit、Visual Studio Code）编写脚本。
    • 例如，创建一个简单的 Python 脚本，列出指定目录中的所有文件：
      python
      import os

directory = "/Users/用户名/Documents"
for filename in os.listdir(directory):
    print(filename)

  3. 运行 Python 脚本

    • 打开 Terminal。
    • 导航到脚本所在的目录：
      bash
      cd /路径/到/脚本目录
    • 运行脚本：
      bash
      python3 script.py

  4. 集成 Python 脚本到 Automator

    • 在 Automator 中，添加 运行 Shell 脚本 操作。
    • 选择 /usr/bin/python3 作为 Shell，并输入脚本内容或调用脚本文件。
    • 保存并运行 Automator 工作流程，自动执行 Python 脚本。

  5. 示例 Python 脚本

    自动备份特定文件夹到外部硬盘

    python
    import shutil
import os
from datetime import datetime

source_dir = "/Users/用户名/Documents/项目"
backup_dir = "/Volumes/备份驱动器/项目备份"

timestamp = datetime.now().strftime("%Y%m%d%H%M%S")
destination = os.path.join(backup_dir, f"项目备份_{timestamp}")

shutil.copytree(source_dir, destination)
print(f"备份完成：{destination}")

11.1.1 系统运行缓慢的优化

问题描述：Mac 电脑在使用过程中变得响应迟缓，应用启动缓慢，系统操作卡顿。

可能原因：

  • 磁盘空间不足
  • 内存占用过高
  • 启动项过多
  • 系统缓存或临时文件堆积
  • 软件或系统更新缺失

解决方法：

  1. 检查磁盘空间

    • 打开 “关于本机” > “存储”，查看磁盘空间使用情况。
    • 删除不必要的文件和应用，清理 下载 文件夹、废纸篓 等。
    • 使用 “优化存储” 功能，自动移除不常用的文件。

  2. 管理内存使用

    • 打开 “活动监视器”（在 应用程序 > 实用工具 中）。
    • 查看 内存 标签，检查哪些应用占用大量内存。
    • 关闭不必要的应用或进程，释放内存资源。

  3. 减少启动项

    • 打开 “系统偏好设置” > “用户与群组” > “登录项”。
    • 查看自动启动的应用，选择不必要的应用，点击 “-” 按钮移除。

  4. 清理系统缓存

    • 打开 Finder，选择 “前往” > “前往文件夹…”，输入 ~/Library/Caches，删除各应用的缓存文件夹。
    • 注意：删除缓存可能会导致部分应用首次启动时稍慢，但不会影响数据。

  5. 更新系统和应用

    • 打开 “系统偏好设置” > “软件更新”，检查并安装最新的 macOS 更新。
    • 打开 App Store，点击 “更新” 标签，更新所有应用。

  6. 重启 Mac

    • 简单的重启可以清理内存和关闭卡顿的进程，恢复系统性能。

11.1.2 应用崩溃与错误修复

问题描述：某些应用在使用过程中频繁崩溃或出现错误提示，影响正常使用。

可能原因：

  • 应用版本过旧或不兼容
  • 应用数据损坏
  • 系统文件损坏
  • 第三方插件或扩展冲突

解决方法：

  1. 更新应用

    • 打开 App Store，点击 “更新” 标签，确保所有应用都更新到最新版本。
    • 对于通过第三方渠道安装的应用，访问开发者官方网站下载最新版本。

  2. 重新启动应用

    • 完全退出崩溃的应用（使用 Command (⌘) + Q 或 活动监视器 强制退出）。
    • 重新启动应用，检查问题是否解决。

  3. 重置应用设置

    • 某些应用允许重置设置或恢复默认配置。
    • 在应用的 “偏好设置” 或 “设置” 中查找相关选项，进行重置。

  4. 删除并重新安装应用

    • 打开 Finder，导航到 应用程序 文件夹，找到问题应用，拖动到 废纸篓。
    • 清空 废纸篓，然后通过 App Store 或开发者网站重新下载并安装应用。

  5. 检查系统文件

    • 使用 磁盘工具 进行磁盘急救：
      • 打开 磁盘工具（在 应用程序 > 实用工具 中）。
      • 选择启动磁盘，点击 “急救”，执行修复操作。

  6. 禁用第三方插件

    • 某些应用的崩溃可能由第三方插件或扩展引起。
    • 打开应用的插件管理界面，禁用所有插件，逐一启用以确定冲突来源。

  7. 查看系统日志

    • 打开 “控制台” 应用（在 应用程序 > 实用工具 中）。
    • 查看崩溃日志，识别导致崩溃的原因或应用，采取相应措施。

12.4 术语汇总

以下是一些在本书中提到的关键术语及其简要解释，供您参考：

术语解释
权限（Permissions）文件或文件夹对不同用户和用户组的访问控制设置，决定谁可以读取、写入或执行文件。
虚拟桌面（Spaces）macOS 的一个功能，允许用户在多个独立的工作空间之间切换，每个桌面可以有不同的应用和窗口。
全屏模式（Full Screen）将应用程序窗口扩展到整个屏幕，隐藏菜单栏和 Dock，提升专注度和工作效率。
分屏模式（Split View）将两个应用程序并排显示在全屏模式下，方便同时进行多任务处理。
沙盒（Sandbox）macOS 的安全机制，将应用程序与系统和其他应用隔离，限制其访问权限，防止恶意软件的入侵。
终端（Terminal）macOS 内置的命令行界面，基于 UNIX，允许用户通过输入命令执行各种系统操作和任务，适合高级用户和开发者。
脚本（Script）一系列自动化命令或指令，通常由脚本语言（如 AppleScript、Python）编写，用于自动执行重复性任务或复杂操作。
加密（Encryption）将数据转换为不可读的格式，只有拥有密钥的人才能解密和读取数据，保护数据的安全性和隐私。
备份（Backup）创建数据的副本，以防止数据丢失或损坏，可以通过 Time Machine、iCloud 或第三方工具实现。
恢复（Restore）从备份中恢复数据或系统状态，确保在数据丢失或系统故障时能够迅速恢复正常使用。
双因素认证（Two-Factor Authentication, 2FA）增强账户安全性的认证方法，除了密码外，还需要第二种验证方式（如短信验证码），防止未经授权的访问。
钥匙串（Keychain）macOS 的密码管理工具，安全地存储和管理用户的密码、证书和其他敏感信息，支持自动填充功能。
快捷指令（Shortcuts）macOS 的自动化工具，允许用户创建自定义的多步骤自动化任务，提升工作效率和操作便捷性。
AutomatormacOS 的图形化自动化工具，允许用户创建工作流程和自动化任务，通过拖放操作简化重复性任务的执行。
App Store苹果官方的应用分发平台，提供各种 macOS 应用的下载和更新，确保应用的安全性和兼容性。
应用程序（Applications）安装在 macOS 系统上的软件程序，用于执行特定的任务和功能，如浏览器、文本编辑器、设计软件等。
系统偏好设置（System Preferences）macOS 的配置中心，允许用户调整系统的各项参数和设置，定制系统行为和外观。
启动项（Login Items）用户登录时自动启动的应用程序和服务，通过 “系统偏好设置” > “用户与群组” > “登录项” 进行管理。
