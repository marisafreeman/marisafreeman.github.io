2. 文件管理

2.1 文件和文件夹管理

创建文件和文件夹：

  1. 创建文件夹：
    • 在 Finder 窗口中，选择一个位置（如桌面或某个文件夹），右键点击空白处。

  2. 创建文本文档：
    • 打开一个文本编辑应用（如 TextEdit）。在 TextEdit 中创建一个新文档，输入内容后，点击左上角的 "File" 菜单，选择 "Save As"（另存为），选择保存位置并为文件命名。

移动、复制和删除文件：

  1. 移动文件：
    • 在 Finder 中，找到你要移动的文件或文件夹。点击并拖动文件到新位置即可。
    • 如果你想移动文件到别的文件夹，只需要拖动文件到该文件夹的图标上即可。
  2. 复制文件：
    • 如果你想复制文件而不是移动文件，可以按住 Option 键，同时拖动文件。此时，文件会被复制到目标位置。

重命名文件或文件夹：

  1. 在 Finder 中，点击选中文件或文件夹，按 Return 键（回车键）。
  2. 输入新名称后，按 Return 键确认即可。

使用 Spotlight 搜索其他内容：

  • 除了文件和文件夹，Spotlight 还可以搜索应用、邮件、网页信息、字典、计算器等内容。例如，输入“calculator”可以快速启动计算器应用。

  2. 打开时间机器设置：
    • 点击屏幕左上角的苹果图标，选择 “System Preferences”（系统偏好设置）。
    • 在设置窗口中，点击 “Time Machine” 图标。
  3. 选择备份磁盘：
    • 点击 “Select Backup Disk”（选择备份磁盘），选择你连接的外部硬盘作为备份目标。
  4. 开启自动备份：
    • 在 Time Machine 窗口中，勾选 “Back Up Automatically”（自动备份）。此时，Time Machine 会自动备份你的文件。

  2. 选择恢复点：
    • 使用屏幕右侧的时间线，选择你想要恢复文件的时间点。
    • 找到文件并点击它，然后选择“Restore”来恢复文件。

    • 在 Apple ID 设置中，选择 “iCloud”，然后勾选 “iCloud Drive”。

  1. 如果你在 Mac 上丢失了某个文件，可以通过 iCloud 恢复文件。打开 iCloud Drive 文件夹，点击文件夹中的 “Recently Deleted”（最近删除）区域查看已删除的文件。

  1. 在 Security & Privacy 中，点击 Privacy（隐私）标签。
  2. 在这里，你可以控制哪些应用有权限访问你的 位置、相机、麦克风、联系人、日历、照片等。
